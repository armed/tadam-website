Tadam is the minimum essence to create a web application with Clojure, simple in its use and professional in results. A Python Flask equivalent.

The rest of the magic will come from you.

Installation

1. Make sure you have Openjdk or Oracle-jdk installed.

Debian/Ubuntu

sudo apt install default-jdk

Mac OS

brew install openjdk

2. Install Clojure and Leiningen.

Debian/Ubuntu

sudo apt install clojure leiningen

Mac OS

brew install clojure leiningen
